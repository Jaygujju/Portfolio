Drive-Thru Queue Optimizer

Discrete Event Simulation Target SLA Performance

Project Overview

An event-driven queuing simulation written in Python. This tool models customer arrival frequencies and order processing lanes using statistical mathematical formulas, simulating bottleneck propagation to find optimal staffing distributions and checkout structures.

The Problem

Quick-service restaurant drive-thrus operate on extremely razor-thin time buffers. During peak morning rushes, a single bottleneck—such as a 10-second delay in POS data entry or coffee brewing—causes immediate traffic backups that spiral lane wait times, leading directly to lost revenue and customer frustration.

What I Did (Technical Action)

Programmed a custom discrete-event simulation model in Python using standard class-based loops. Utilized exponential probability functions (`random.expovariate`) to simulate customer arrivals (Poisson process) and customer service times. Created a state machine to track lane queues, calculate cumulative order-to-delivery wait times, and run multi-run profiling trials to output average latency scores.

Modeled front-line drive-thru efficiency criteria based on my experience as a Team Member at Tim Hortons. Analyzed Point-of-Sale (POS) interactions, order taking, and assembly stages. Programmed a target 45-second service-window SLA constraint, simulating checkout bottleneck factors to discover how shift scheduling and station coordination impact overall lane capacity.

Why This Stands Out

It demonstrates mathematical and algorithmic modeling capabilities. It shows I can apply probability theory (Poisson queues) and state-machine logic to real-world queuing issues, analyzing bottlenecks through simulation profiles.

Proves a relentless focus on customer satisfaction and target SLAs. It shows I understand that modern retail success relies on micro-optimizations (like cutting transaction times by 10s) and can use software to model physical workflow scheduling under pressure.

Discrete-Event Queue Simulator

queue_sim.py
import random
import math

class DriveThruSimulator:
    def __init__(self, arrival_rate, service_rate):
        self.arrival_rate = arrival_rate # Avg customers arriving per minute
        self.service_rate = service_rate # Avg customers served per minute
        self.current_time = 0.0
        self.queue_length = 0
        self.total_waiting_time = 0.0
        self.completed_customers = 0

    def run_simulation(self, total_duration_minutes):
        # 1. Schedule initial customer arrival using Poisson distribution
        next_arrival = random.expovariate(self.arrival_rate)
        next_service = float('inf')
        
        while self.current_time < total_duration_minutes:
            # 2. Determine which event happens first: Arrival or Checkout completion
            if next_arrival < next_service:
                # --- CUSTOMER ARRIVAL EVENT ---
                self.current_time = next_arrival
                self.queue_length += 1
                
                # If server is idle, immediately start checkout transaction
                if self.queue_length == 1:
                    next_service = self.current_time + random.expovariate(self.service_rate)
                    
                # Schedule subsequent arrival
                next_arrival = self.current_time + random.expovariate(self.arrival_rate)
            else:
                # --- CUSTOMER SERVICE COMPLETION EVENT ---
                self.current_time = next_service
                self.queue_length -= 1
                self.completed_customers += 1
                
                # Schedule next checkout transaction if customers remain in lane
                if self.queue_length > 0:
                    next_service = self.current_time + random.expovariate(self.service_rate)
                else:
                    next_service = float('inf')

        return self.completed_customers

# Initialize: 1.5 arrivals/min (rush hour), 2.0 checkouts/min (30s avg service)
sim = DriveThruSimulator(arrival_rate=1.5, service_rate=2.0)
cars_completed = sim.run_simulation(total_duration_minutes=60.0)

print(f"🚀 Drive-thru simulation run complete.")
print(f"Cars Served in 1 Hour: {cars_completed} vehicles")
print(f"Average Lane Congestion: {sim.queue_length} vehicles remaining")
